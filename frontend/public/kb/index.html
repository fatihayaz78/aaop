01Platform Overview

AAOP (Agentic AI Observability Platform) is an AI-powered OTT platform observability system built under the Captain logAR brand. The primary tenant is S Sport Plus (aaop_company). The platform follows a local-first architecture with a clear GCP migration path via the adaptor pattern — all local data stores (SQLite, DuckDB, ChromaDB, Redis) map 1:1 to GCP-managed equivalents without touching agent code. 08Startup Commands

# Activate virtual environment
source ~/.venvs/aaop/bin/activate

# Start backend (FastAPI — port 8000)
python -m uvicorn backend.main:app --reload --port 8000

# Start frontend (Next.js — port 3000)
cd frontend && npm run dev

# Start Redis
redis-server

# Health check
curl http://localhost:8000/health
Important: Always use python -m uvicorn (not just uvicorn) to ensure the correct virtual environment Python is used.

09Red Lines

✗print()→ Use structlog for all logging ✗Python 3.14→ Stick to Python 3.12 ✗kaleido 1.x→ Pin to kaleido==0.2.1 (Chrome dependency) ✗Direct DB access inside Agent→ Always go through tools ✗HIGH risk tool auto-run→ Set approval_required=True ✗Sending raw PII to LLM→ Run through pii_scrubber.py first ✗Hardcoded credentials→ Use .env file ✗Tool call without tenant_id→ All tools require tenant_id ✗Undocumented ADR→ Update ARCHITECTURE.md

10Local → GCP Migration Map

Local GCP Equivalent Adaptor File What Changes
SQLite Cloud Spanner shared/clients/sqlite_client.py Adaptor only
DuckDB BigQuery shared/clients/duckdb_client.py Adaptor only
logs.duckdb BigQuery shared/clients/logs_duckdb_client.py Adaptor only
ChromaDB Vertex AI Vector Search shared/clients/chroma_client.py Adaptor only
asyncio.Queue Google Cloud Pub/Sub shared/event_bus.py Adaptor only
Redis (local) Cloud Memorystore shared/clients/redis_client.py Host changes only
.env secrets GCP Secret Manager shared/utils/settings.py Provider changes
Key principle: Agent code (apps/*/agent.py, apps/*/tools.py) does NOT change during GCP migration. Only the adaptor layer inside shared/clients/ is swapped.
